# 抢课脚本使用教程

本目录当前有两个可用脚本：

- `grab_course.py`：单门课搜索后持续抢课
- `grab_course_leak_any9_manual.py`：预设 9 门课，任意抢到 1 门即停止

本文先讲单课版；9 门捡漏版请看 [使用教程-9门捡漏版.md](/Users/lizhaohui/Desktop/抢课脚本/使用教程-9门捡漏版.md)。

## 一、运行前准备

建议用同一个 `python3` 安装依赖和运行脚本，避免环境不一致。

```bash
python3 -m pip install playwright
python3 -m playwright install chromium
```

如果 `python3` 不是你想用的解释器，先执行：

```bash
which python3
python3 --version
```

## 二、脚本文件

单课版运行命令：

```bash
cd ~/Desktop/抢课脚本
python3 grab_course.py
```

## 三、运行前需要确认的配置

打开 [grab_course.py](/Users/lizhaohui/Desktop/抢课脚本/grab_course.py)，重点确认这几个变量：

```python
BATCH_ID = "1b380a657c11450aa9dc99586618643b"
BATCH_BEGIN = "2026-03-31 12:00:00"
RETRY_INTERVAL = 0.5
SEARCH_PAGE_SIZE = 100
RATE_LIMIT_COOLDOWN = 5.0
HTML_RESPONSE_COOLDOWN = 8.0
```

含义：

- `BATCH_ID`：当前选课批次 ID
- `BATCH_BEGIN`：该批次开放时间
- `RETRY_INTERVAL`：抢课重试间隔
- `SEARCH_PAGE_SIZE`：搜索课程时每页拉取数量
- `RATE_LIMIT_COOLDOWN`：遇到“请求过快”后的冷却时间
- `HTML_RESPONSE_COOLDOWN`：接口返回 HTML 页面时的冷却时间

如果学校切了新的批次，通常最先要改的是 `BATCH_ID` 和 `BATCH_BEGIN`。

## 四、完整使用流程

### 1. 启动脚本

```bash
cd ~/Desktop/抢课脚本
python3 grab_course.py
```

脚本会弹出 Chromium 浏览器，并打开选课系统登录页。

### 2. 手动登录

- 在浏览器中完成登录和验证码
- 回到终端按回车

脚本会读取当前浏览器里的 `token`。如果终端提示未检测到 `token`，说明登录还没完全成功，先回浏览器确认已进入学生信息页或选课页。

### 3. 搜索课程

终端会提示输入关键词。支持：

- 输入课程名关键词
- 输入教师名关键词

脚本会先按课程名搜索；如果没有结果，再按教师名搜索。

### 4. 选择目标课

搜索结果会显示：

- 课程名
- 教师
- 时间地点
- 余量
- `JXBID`

按终端提示输入序号即可开始抢。

### 5. 自动抢课

如果还没到开放时间，脚本会自动等待到 `BATCH_BEGIN` 再开始。

开放后脚本会持续循环请求，直到：

- 抢课成功
- 你手动停止
- 或出现需要人工处理的异常

如果系统返回 `301` 二次确认，脚本会自动再次提交。

## 五、常见情况

### 1. 搜索不到课程

- 先确认 `BATCH_ID` 是否正确
- 换课程名或教师名再搜一次
- 确认该批次和当前账号确实有权限看到这门课

### 2. 一直提示“请求过快”

脚本会自动冷却再继续。如果频繁出现，可把 [grab_course.py](/Users/lizhaohui/Desktop/抢课脚本/grab_course.py) 里的 `RETRY_INTERVAL` 适当调大一些。

### 3. 接口返回 HTML 页面

这通常表示：

- 被打回登录页
- 命中风控页
- 或临时网关异常

脚本会自动冷却并刷新；如果仍异常，就需要你重新登录。

### 4. 登录失效

如果运行中会话过期，脚本会提示你重新登录。按提示在浏览器完成登录后，回终端继续即可。

### 5. 浏览器没弹出来

先执行：

```bash
python3 -m playwright install chromium
```

如果还是不行，确认当前 `python3` 和安装 Playwright 时使用的是同一个解释器。

## 六、停止脚本

中途想停止时，在终端按：

```bash
Ctrl + C
```

---

## 七、使用建议

- 提前几分钟启动脚本并登录
- 浏览器窗口不要随意关闭
- 终端保持打开，不要让电脑休眠
- 抢到后立刻回教务系统确认最终结果

---

## 八、补充说明

- 浏览器配置目录由脚本内部指定，主要用于保存当前会话
- 选课结果以教务系统最终显示为准
- 本脚本只是加快正常操作，不会绕过账号权限或开放时间限制
